About This Site

As a graduate student at the UNIST NLP Lab, I began Conference Deadlines as a personal project to solve a simple problem: keeping track of the ever-growing list of conference deadlines. I decided to open-source the project to help graduate students and researchers stay organized and focused on their work.

This website is a fully static, client-side application built with HTML5, Bootstrap, and vanilla JavaScript, and it's hosted on GitHub Pages. All data is managed through a simple 'conferences.json' file, all with the goal of keeping it lightweight, fast, and easy to maintain. I got a lot of help from Google's Gemini throughout the development process.

How to Add Conference

We Welcome contributions to keep our deadline list accurate and comprehensive. If you want to add or update a conference, you have two options:

  1. Submit a Pull Request on GitHub: Fork the repository, edit the 'conferences.json' file, and submit a pull request with your changes. This is the fastest way to contribute.
  2. Send me an email: If you're not familiar with GitHub, simply send an email to mindw96@unist.ac.kr with the official website link for the conference you'd like to add.

Our Lab

Here is the lab I'm a part of. Please get in touch if you find our research interesting!

The UNIST NLP Lab is developing System 2-based language AI that thinks and learns like humans. We're going beyond simple recall to build AI with real understanding, reasoning, and learning capabilities.

Our current research focuses on:

  • Editing and leveraging knowledge in unstructured text
  • Efficient reasoning based on knowledge learning
  • Integrating external memory with parameter-efficient LLMs
  • Progressive knowledge expansion via Mixture-of-Experts (MoE)

We're building the foundational technologies for next-generation language intelligence that can truly learn and adapt like humans do.
